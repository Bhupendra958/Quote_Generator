# 💬 Quote Generator

A modern and responsive Quote Generator built using **HTML, CSS, and JavaScript**. The application fetches random inspirational quotes in real time from the **DummyJSON API** and displays the quote along with the author's name. Users can generate a new quote instantly using the Next and Previous buttons.

## 🚀 Features

- 💬 Generate random quotes
- ✍️ Display quote author
- 🔄 Load a new quote instantly
- 🌐 Fetch real-time data from an API
- 📱 Responsive and modern UI
- ⚡ Fast and lightweight
- 🎨 Glassmorphism design

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Fetch API
- DummyJSON Quotes API

## 📖 How It Works

1. The application loads a random quote when the page opens.
2. JavaScript sends a request to the DummyJSON Quotes API.
3. The API returns a random quote and author.
4. The quote and author are displayed inside the quote box.
5. Clicking **Next** or **Previous** loads another random quote.
